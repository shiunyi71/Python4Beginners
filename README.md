## 📌 誰適合參加這場程式冒險之旅

🎮 **遊戲新手玩家** - 從未碰過程式，但想用Python打造屬於自己的小遊戲和工具  
🧩 **邏輯解謎愛好者** - 喜歡拆解問題，想學會用程式語法寫出聰明的解決方案  
🔍 **好奇心探險家** - 對電腦如何「聽懂」人類指令感到好奇，想掌握基本程式邏輯  
⚡ **創意實作達人** - 想把腦中的想法變成實際可以運行的程式，體驗創造的樂趣  
🎯 **生活應用實踐者** - 希望學會基礎語法後，能設計溫度轉換器、小遊戲等實用工具  

❌ **不適合的夥伴**：  
- 覺得看到紅色錯誤訊息就會害怕的人（我們把bug當作解謎遊戲！）  
- 認為一定要先背完所有語法才能開始的人（邊玩邊學才是王道！）

💡 **我們的學習方式**：透過趣味專案學語法，用遊戲化方式掌握程式邏輯，讓每一行程式碼都充滿樂趣！

## 課程學習目標：

1. **掌握基礎程式設計思維：** 學習程式設計的基本邏輯與思考方式，培養運算思維與問題拆解能力
2. **精通Python基礎技能：** 熟練掌握Python基本語法和資料結構，能運用條件判斷與迴圈解決問題
3. **培養實作與應用能力：** 能獨立開發簡單的應用程式，會運用Python解決生活中的實際問題
4. **建立持續學習的態度：** 培養自主學習程式設計的興趣，建立解決程式問題的信心

## 📌 課程重點

### 🔹 學習評量方式
本課程採用以下評分標準：
- **實作評量** 作業與期末專題之繳交分數 (70%)。
    - 作業評量：40%
    - 期末專題評量：30%
- **出席狀況** (20%)
    - 課堂出席與參與情形評定，由課堂教師給予評分。
- **上課表現** (10%)
    - 包含課堂參與、討論表現、團隊合作等，由課堂教師給予評分。

**※評分標準依實際課程進行調整，大學端70%僅為建議，最終以課堂教師評定為準。**
